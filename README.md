# Orçamento

Esse programa foi feito para simular um orçamento de aluguel de forma simples. A ideia é ajudar o usuário a escolher o tipo de imóvel, como apartamento, casa ou estúdio, e ir respondendo algumas perguntas para montar o valor final do aluguel. Dependendo das escolhas, como quantidade de quartos, наличие de garagem ou vagas, o preço vai sendo ajustado automaticamente. No caso do apartamento, por exemplo, ainda existe um pequeno desconto se a pessoa não tiver filhos, o que mostra uma regra de negócio aplicada no código.

Depois de calcular o valor mensal, o sistema também simula o contrato, que tem um valor fixo e pode ser parcelado em até cinco vezes. O programa faz essa divisão e mostra quanto fica cada parcela, deixando tudo bem claro para o usuário.

Por fim, ele gera um arquivo CSV com os valores do aluguel para os 12 meses do ano. Isso é útil para ter um controle ou até abrir no Excel depois. A navegação é toda feita pelo terminal, com perguntas diretas e respostas simples, então qualquer pessoa consegue usar sem dificuldade.
